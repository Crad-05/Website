    Ríomhaire was founded with the goal of providing a new Linux that could delivered on multiple desktop environments, with still having the versatility of other Linux distro's whilst eliminating the more inconvienent aspects that can come from Linux based Operating systems making it accessible for new users to Linux with the power of open source freedom.

    Ríomhaire Linux is a new Arch based Linux, based of 6.1 kernel that provides the latest Linux experience, in 3 flavours GNOME, KDE and XFCE desktop environments. Providing an efficient desktop for Productivity, office work and gaming. That empowers you with the power open-source software.

    Latest LTS is Ríomhaire V1.02
